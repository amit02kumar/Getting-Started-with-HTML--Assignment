HTML Tags:- 
Tags are the starting and ending part of an html element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags.
There are two type of tags container tag and empty tag. container tags are the tags that have both opening tag as well as closing tag ex- <p> </p> , <h1> </h1> and empty tags are the tags that only have opening tag ex- <br>, <hr>.

HTML Element:-
Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag. ex- <h1>This is a heading </h1>.
